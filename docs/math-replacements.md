# Math replacements

Bitbob features a plethora of `Math` functions to replace when calculating 32bit integers for better performance. Those include:

| `Math`      |  bitbob                           |
|-------------|-----------------------------------|
| `Math.max`  | `max`                             |
| `Math.min`  | `min`                             |
| `Math.abs`  | `abs`                             |
| `Math.sign` | `sign`                            |
| `Math.log2` | `intLog2`                         |
| `Math.log10`| `intLog10`                        |

There are also a few methods working on floats.

| `Math`      |  bitbob                           |
|-------------|-----------------------------------|
| `Math.ceil` | `ceil`                            |
| `Math.floor`| `floor`                           |
| `Math.round`| `round`                           |
